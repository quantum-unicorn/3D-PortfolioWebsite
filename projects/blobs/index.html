Project "LucidBlobs"

Procedurally generated collection of 10 000 unique blob creatures

Originally concieved as an NFT collection, this project was an experiment in procedural generation using the Blender geometry nodes system. Each blob is comprised of randomly picked colors, material, modifiers and accessories.

Furthermore, I wrote a custom smart contract, minting dapp and created an Opensea test collection. Due to lack of interest, the collection was never released.

Procect "Voice-VAE"

Neural network for voice compression

This deep, convolutional neural network uses the variational autoencoder architecture to compress human speech to a 32-dimensional latent space and decode it back to frequency space. Further research may be able to use this as a base for a voice changer, which was the initial inspiration for this project.

The model consists of an encoder of 5 1D convolutional layers and a symmetrical decoder. The total number of parameters is currently close to 850k but could probably be optimized. The model was built using PyTorch trained on the Mozilla Common Voice dataset.

Figurine clone

3D printed, 2x scale clone of an angel figurine

On this project, the assignment was to 3D print a 2x scale clone of a figurine of an angel.

I used photogrammetry to create a base mesh of the reference figure,

I then used Blender to postprocess the base mesh using sculpting tools,

and finally 3D printed the model in resin.

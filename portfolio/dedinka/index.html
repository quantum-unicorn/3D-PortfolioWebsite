ArchViz Study

An ArchViz study for a construction company

The goal of this project was to create photorealistic renderings of an apartment building.

I was provided with the architect’s line renderings but no 3D files. I modelled the building from this reference and built out the surrounding area using photogrammetry and image projection to get accurate lighting and reflections.
